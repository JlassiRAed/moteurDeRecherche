 The British Rail Class diesel mechanical shunter shunting locomotive class was built between and It was the basis for the later British Rail Class Class built in the British Railways workshops The Class locomotives were supplied by the Drewry Car Co which at the time had no manufacturing capability Drewry sub contracted the construction work to two builders Both of these built other locomotives under the same arrangement Early locomotives were built by Vulcan Foundry and later examples were built by Robert Stephenson and Hawthorns
