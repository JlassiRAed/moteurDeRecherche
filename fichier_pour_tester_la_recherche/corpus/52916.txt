Links text word by word The link target defaults to Wiktionary If you don t want to link to Wiktionary entries set pref equal to some other value such as your favorite interlanguage prefix or c for Commons Use the arguments n n n etc to specify items that should not be linked Regular spaces will be removed from the beginning and end of these if you must have spaces use amp nbsp instead If you want to have a delimiter such as amp nbsp appear between links specify the delimiter with lim If desired you can specify the language code with lang Examples includeonly
