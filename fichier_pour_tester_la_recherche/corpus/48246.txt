This template returns the number of full years and surplus days between two specified dates If the second set of parameters is not included it will return the number of full years and surplus days between a specified date and today s date The two numbers are separated by a comma the words year and day are plural when appropriate The template is included in the year value so that values produced by this template will work properly in sortable lists See Wikipedia Age calculation templates for more information on using this kind of template Syntax or Examples When using this template to calculate a person s age at death or a person s age on a specified date please substitute it into the page Example Note This template does not check for incorrect input Note also that due to leap years the measure years and days fails to be fully additive The following example is not an error but each calculation uses a different definition of year either days or but split up
