The Wikipedia software recognizes inline PMID numbers The software automatically creates a search link to PubMed in the same way that it does for International Standard Book Number s How Wikipedia Works and How You Can Be a Part of It p No template is required to enable this feature When a PMID is known it may be helpful to use it in the same way as it is good practice to use ISBN references for book sources It is important when making a link to have only a space after PMID followed by the numbers do not use the colon which is present in PubMed citations Example Correct Incorrect PMID Incorrect PubMed The formatted citation is displayed as a blue link
