The TIME service is an Internet Protocol Internet protocol defined in RFC Its purpose is to provide a site independent machine readable date and time TIME can operate over either Transmission Control Protocol TCP or User Datagram Protocol UDP When operating over TCP a host connects to a server that supports the TIME protocol on TCP port The server then sends the time as a bit unsigned binary number in network byte order representing a number of seconds since January GMT and closes the connection The host receives the time and closes the connection When operating over UDP the client sends a datagram to UDP port The server responds with a single datagram of length containing the time There is no connection setup or teardown
