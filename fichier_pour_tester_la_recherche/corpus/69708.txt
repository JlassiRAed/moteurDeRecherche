 Wikitravel is a project dedicated to creating a travel guide reliable current complete and open content Although it is based on wiki wiki technology the project is intended to be used to create printed guides Wikitravel is created with the help of Wikitravellers in worldwide Wikitravel was created in July by Evan Prodromou and Michele Ann Jenkins It is inspired in part by Wikipedia The project uses MediaWiki like the Wikipedia but it is not a project of the Wikimedia Foundation Wikitravel formerly used a Creative Commons Attribution ShareAlike but since January uses a Creative Commons Attribution ShareAlike Among other things this facilitates the reprinting of pages
