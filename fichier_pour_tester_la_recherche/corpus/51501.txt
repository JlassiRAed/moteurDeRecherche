The Precision Approach Path Indicator is a visual system next to the touch point on a runway The purpose of the system is to lead a plane in correct approach The PAPI creates ray of light which gives pilot information about his actual height Principle If plane is over correct approach line pilot will see white lights and red If he is higher he will see all lights in white If plane keeps correct approach pilot will see white lights and red The right line is in the angle degrees from axis of airport If plane is under correct approach line pilot will see red lights and white If he is lower he will see all lights in red The rules about the parameters are written in standard ANNEX ICAO The brochure contains more specific information about intensity of ray location where we can get device etc File PAPI M jpg The PAPI deviceFile PAPI german svg Principle of PAPI indication
