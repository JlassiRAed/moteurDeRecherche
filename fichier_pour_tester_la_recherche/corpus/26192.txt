 GNU Emacs is a text editor that is common on many UNIX based operating systems Emacs is primarily used by programmer programmers Emacs is made powerful by Emacs Lisp a built in programming language that lets the user extend the capabilities of the editor A common Emacs joke is that all of the functions of the editor are crazy weird keystrokes In reality though these keystrokes are relatively simple though they can take some getting used to There is an Internet turf war between programmers that prefer Emacs and programmers that prefer Vim another common text editor
