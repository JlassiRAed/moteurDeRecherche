 NEXTSTEP was a computer operating system made by a company called NeXT NeXT was run by Steve Jobs who was the CEO of Apple Inc Apple until his death in NEXTSTEP was based on UNIX and used source code from Berkeley Software Distribution BSD It had a graphical user interface and it let people write computer program s using object oriented programming NEXTSTEP ran only on computers made by NeXT Later on NEXTSTEP was changed so that it could run on other computers This new operating system was called OPENSTEP In Apple bought the NeXT company and used NEXTSTEP to make Mac OS X
