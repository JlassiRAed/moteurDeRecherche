 Endianness refers to how hexadecimal data is ordered in machine language the simplest most understandable code that a computer can use In computer coding certain numbers usually two bytes long that are called words can be written or input in two ways little endian and big endian It depends on the data format and the way the computer reads the data Say that we have a word like this AB Now the way it s written right now is big endian because the bigger number is on the end To write it in little endian we simply switch the positions of the first two and the last two pieces so it becomes AB AB
