In computing or computer an image scanner is a device used to transfer image s or text into a computer There are special models for scanning photo negatives or to scan books In the computer the signal from the scanner is transferred to a digital image This image can then be edited printed etc All scanners have special parts which are used to take a picture from an object Charge coupled device or Contact Image Sensor parts take the light from the object and change it into a digital signal The digital signal is then transferred to the computer memory or is read in the scanner s processor With some software it can be changed to a compressed digital image like JPEG or PNG Text written by hand can be changed to ASCII code by an Optical character recognition OCR program It is very important for computer
