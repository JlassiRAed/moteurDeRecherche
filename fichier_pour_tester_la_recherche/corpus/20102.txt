 BeatrIX is a GNU Linux live CD based Linux distribution started by Steven Watsky it was designed to look very simple in order to be as easy to use as possible to new users A new BeatrIX desktop has just four icons Hence anyone who has some computer experience will be able to use BeatrIX BeatrIX is a small operating system taking up only MB The liveCD fits into mm discs that can store up to MB of data BeatrIX uses Linux and GNOME as its GUI desktop It has productivity tools such as OpenOffice org Mozilla Firefox etc BeatrIX has been discontinued since Another distro called BeaFanatIX is using the code of BeatrIX and continuing development
