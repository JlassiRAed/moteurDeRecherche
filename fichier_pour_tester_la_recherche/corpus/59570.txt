Usage basic usage used with all optionsAll parameters are optional Country name This is the name that is shown for the country For example USA instead of United States or UK instead of United Kingdom If no name is given the base name of the page is used Alternate flag This is the name of the flag to be shown If no flag is given the flag will be based on the name of the page The flag is chosen based on the Template Country alias and Template Country flag alias for which ever country this template is told to use It uses the same functions as the template to choose the flag includeonly
