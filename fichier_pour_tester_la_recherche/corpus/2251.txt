 Linear B is an ancient script used to write Mycenaean Ancient Greek language Greek the oldest known form of Ancient Greek This was proved by Michael Ventris in the early s Linear B came before the Greek alphabet by several centuries The script is based on Linear A a script which cannot be read today The oldest texts written in Linear B date from about BC It disappeared with the fall of the Mycenaean civilization There are about signs in Linear B that represent syllable s In addition there are many ideogram ideographic signs These represent objects or commodities They do not represent sounds and are never used as word signs in writing a sentence Linear B seems to have only been used for administration In all the thousands of clay tablets a relatively small number of different hands The script may have only been used by a guild of professional scribe s These scribes worked at the different palaces when the palaces were destroyed the script disappeared
