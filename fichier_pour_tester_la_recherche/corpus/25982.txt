A workstation is a computer used for scientific or technical calculations or purposes These computers are usually expensive high end computers They are different from the ordinary computers used for text processing Very often they have operating system s that allow several users to work at the computer at the same time Very often workstations are connected to a computer network Computer terminal s may be used to access them Currently almost all workstations use a unix or unix like operating system Workstation computers often use a bit server class Central processing unit CPU such as the Intel Xeon or AMD Opteron and some have more than one CPU They may also have more memory than a normal desktop computer and they often have or RPM hard drive s or solid state drive s Some workstations have devices that are not commonly used on normal desktop computers such as ECC memory SCSI fibre channel and gigabit Ethernet Some technologies that were once only found on workstations have now become common in normal PCs such as multi core processors solid state drives hardware accelerated graphics bit color broadband Internet access virtual memory pre emptive multitasking and OpenCL Common uses for workstations include Computer aided technologies computer aided design D graphics and animation video and audio production weather modeling and other Computer model simulation s and scientific work
