 NTFS is a file system It was developed by Microsoft A file system is a way of storing computer files NTFS stands for New Technology File System Microsoft introduced it with the Windows NT operating system A version of NTFS is also included in the most recent operating systems developed by Microsoft such as Windows This was first seen in the operating systems Windows XP Windows Server and Windows Vista NTFS was made to replace the old file system used by earlier versions of Windows like Windows Windows and Windows SE That file system was also used by MS DOS and is called File Allocation Table FAT
