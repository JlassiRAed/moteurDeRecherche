 The Game port is a computer port port that is present on some Personal Computer PC hardware and hardware similar to it This port lets the connection of analog signal analog devices used to control computer game s such as joystick s In most relevant hardware designs for personal computers in the port is an integrated component on the sound card another Input output I O card or the motherboard itself Starting with Windows Vista Microsoft no longer supports this port with drivers from the Operating System but third party drivers are still available There are also drivers that allow to map the game port to an Universal Serial Bus USB device
