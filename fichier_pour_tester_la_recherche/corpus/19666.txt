 The Game Boy Camera is an accessory for the Game Boy line of video game consoles It was released in Japan North America and Europe in It allows a person to take x black and white digital pictures It is meant to be used with the Game Boy Printer which uses a special kind of paper to print saved pictures It can be used with all Game Boy consoles except for the Game Boy Micro It was the only Nintendo product to allow a person to take pictures until the release of the Nintendo DSi console
