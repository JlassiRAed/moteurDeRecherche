This is the or multi namespace message box meta template This meta template is used to build message box templates that are used on several types of pages and thus need to change style depending on what page they are used on Based on page type detected it uses one of Note that this template should only be used for message boxes that really need to adapt their style Most message boxes do not need this and should use one of directly Using those templates directly means that your template will look the same on its template page and at any other place you show it which makes it clear on what kind of pages it is supposed to be used It also gives you access to any extra features those templates offer and it saves some server load
