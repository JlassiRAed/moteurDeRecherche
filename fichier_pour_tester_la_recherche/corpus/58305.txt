This template is for optionally sorting tables by specified sortkeys If necessary one can apply sorting using a sortkey which Help Cascading style sheets Non display due to CSS is not displayed Javascript sorting is based on the text inside and outside the tags without the tags themselves A hidden sortkey can be put at the start This template makes that more convenient The first parameter is the sortkey the second the wikitext of what has to be displayed By default this is the sortkey with link brackets To use it place this in your table cell It presently works through hidden CSS as described at Help Sorting If a cleaner way of doing this is developed the template will be changed A table using this template should use it in all rows
